# Student Productivity Manager

This project is a productivity manager designed for students to organize their tasks, assignments and goals.

I created this application as part of my programming portfolio to practice full-stack development and database integration.

## Project Idea

As a student myself, I often manage many assignments, deadlines and responsibilities.  
This application helps organize tasks and track productivity in a simple dashboard.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- View pending and completed tasks
- Store tasks in a database

## Technologies Used

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js

Database
- MySQL

## How It Works

Users can create tasks and store them in a database.  
Each task contains:

- Title
- Description
- Due date
- Status

The application retrieves tasks from the database and displays them on the dashboard.
